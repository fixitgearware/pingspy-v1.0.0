  <h4>gitclone https://github.com/fixitgearware/pingspy-v1.0.0.git </h4>

  <h4>cd pingspy-v1.0.0 </h4>

- <h2>Using The Help Commands :</h2>

  <h4>python3 main.py --help </h4>
